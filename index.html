Metaprogramming

Códigos criados/modificados em tempo de execução 

class Book

    def initialize(title)
        @title = title
    end

    def title
        @title
    end

    def title=(title)
        @title = title
    end
end

Metaprogramming

É o mesmo que: 

class Book

    attr_accessor :title

    def initialize(title)
        @title = title
    end

end
